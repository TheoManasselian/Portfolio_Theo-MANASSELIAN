Présentation du projet

Carnet de compte bancaire numérique fonctionnant entièrement en local sur la machine. L'application sert de carnet familial : elle s'ouvre dans le navigateur via un petit serveur Node.js et ne dépend d'aucun service en ligne ni d'aucune base de données externe.

Ce projet personnel m'a permis de construire un serveur Node.js de bout en bout sans framework : gestion manuelle des routes, d'une petite API JSON et du service des fichiers statiques, avec une persistance par fichiers pensée pour rester facile à sauvegarder et à transporter.

Fonctionnalités

  • API maison — routes GET / POST pour lire et écrire les données du mois
  • Vue mensuelle — un sélecteur permet de naviguer d'un mois à l'autre
  • Stockage local — un fichier JSON par mois dans le dossier data/
  • Nouveau mois automatique — le fichier du mois courant est créé au démarrage
  • Portable — copier le dossier suffit pour changer de PC
